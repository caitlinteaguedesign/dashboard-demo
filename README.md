# Dashboard Demo
Created for the IT department of an ecommerce company to monitor real-time business and technology metrics. This project is the front-end experience that served data from various API (developed by others for this app to consume). 

This app is a "demo" of the original app. Placeholder data is substituted for real data, and functions like `fetch()` may be commented out or no longer working.

## Notes
- Originally bootstrapped with Create React App, and migrated to Vite in 2026.
- Keep chart.js in major version 2. Breaking change in latest release.
